Sector List implementation and examples

Sector List implementation with interative examples

This page presents supplementary materials to the paper entitled "Representing and processing polygonal maps with Sector Lists". Sector Lists (SLs) is a vector-based representation scheme for polygonal maps, where each point of the plane is mapped to a single scalar value.

SLs represent polygonal partitions by their boundaries, but rather than structuring these boundaries as circular lists of vertices, all regions are modeled as a set of special objects called sectors. The sectors are organized in lists for efficient plane sweep algorithms using a differential approach.

SLs contrast with usual polygon-based representations since the latter require more coarse-grained structures such as multipolygons, circulations and holes, as well as must enforce only one value per point.

The following are four sections for understanding and interacting with the characteristics and properties of the Sector List.

  • 1) Sector List Implementation in JavaScript: data structure with the operations and algorithms described in the paper.
  • 2) Sector List Examples: interactive explanations presenting the fundamental operations.
  • 3) Sector List Experiment 1 - Map Algebra: exemplifies the flexibility of Sector Lists applying Sector Lists to perform map algebra and GIS multi-criteria analysis.
  • 4) Sector Lists Experiment 2 - Random Polygons: aims at assessing the computational efficiency of Sector List operations as implemented in our prototype, as well as providing some empirical evidence for the complexities of the algorithms.
